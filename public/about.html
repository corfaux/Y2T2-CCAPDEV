    About LabSys

    A Laboratory Reservation System for DLSU students based on the booking system presently enforced through LibCal for the Henry Sy Discussion and Viewing Rooms.

    System Overview

    Student users are allowed to reserve up to 3 seats in 30-minute time intervals from 7am to 5pm, only within the current Monday to Saturday working week. They are also able to customize their profile, visible only to the users themselves and lab technicians for identification and data collection. Lab Technicians possess administrator privileges and are able to manually create and cancel reservations, input new buildings, laboratories, and view student profiles.

    Student User Features

    • Create reservations at any laboratory (Max 3 Seats and 6 time slots)
    • Edit reservations
    • Upload profile picture
    • Edit profile information

    Admin User Features

    • Add/Edit/Delete Laboratories
    • View/Delete Reservations
    • View Student Profiles
    • Create New Reservations
    • Search reservations within a specified date and lab.

    Languages/Platforms Used

    • Frontend: HTML, CSS, JavaScript
    • Backend: Node.js, Express.js
    • Database: MongoDB Atlas
